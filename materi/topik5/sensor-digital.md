# Sensor Digital

- [Sensor Raindrop](https://raspi.tv/2017/make-a-rain-alert-system-with-raspberry-pi)
- [Humidity Sensor (DHT11)](https://tutorials-raspberrypi.com/raspberry-pi-measure-humidity-temperature-dht11-dht22/)
- [PIR Motion Sensor](https://tutorials-raspberrypi.com/connect-and-control-raspberry-pi-motion-detector-pir/)
- [Ultrasonic Sensor (HC-SR04)](https://tutorials-raspberrypi.com/raspberry-pi-ultrasonic-sensor-hc-sr04/)
